Objectifs de cette semaine

Vous êtes une équipe de "data-scientist".
Votre entreprise souhaite développer des fonctions avancées sur le site d'e-commerce.

  • sélection d'article
  • sélection multi-critère avec meilleurs compromis
  • recherche de produit similaire
  • moteur de recommandation

Fonctionnalités avancées

  1. sélection multi-critère avec meilleur compromis => Proposer plusieurs critères (prix, caractéristiques, disponibilité, notation) et être capable de proposer les meilleurs alternatives si aucun produit ne correspond
  2. recherche de produit similaire => si un article n'est plus disponible trouver l'article le plus proche
  3. moteur de recommandation => à partir d'un panier, proposer les articles que la personne est la plus susceptible d'acheter
  4. Recherche personnelle => Comment fait-on fonctionnellement ?

Réponse technique

Vous êtes dans un cadre où vous êtes une équipe et votre produit doit être intégré dans une autre équipe.
