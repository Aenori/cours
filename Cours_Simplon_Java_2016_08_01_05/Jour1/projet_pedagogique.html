II. Planning général

Jour 1 : introduction, algorithmie de base, manipulation de listes

Petite introduction type jeu de rôle "vous êtes une équipe de data-scientist ...", petite introduction sur les différents types d'intégration pour justifier le fait de faire un programme "from scratch".
Algorithmie de base sur des objets simples (3, 4 attributs) notamment sur les prix : sélection de l'article le plus cher, le moins cher, d'une liste d'article correspondant à une fourchette de prix, recherche d'articles ayant le même prix.
Comment accélérer ces algorithmes ?? Implémentation d'une recherche dichotomque, ré-implémentation des exercices précédents sur une liste triée, algorithmes de tri pour les plus avancés.

Jour 2 : test unitaire, manipulation de fichiers xml

Matin : développement de tests unitaires pour les fonctions précédentes avec JUnit
Après-midi : lecture de fichiers XML, intégration dans les fonctionnalités précédentes

Jour 3 : suite de la manipulation de fichiers xml, xml relationnels

Matin : suite de la manipulation sur les fichiers XML
Après-midi : XML relationnel, une liste de promotion sur les produits est fournie en entrée, il faut en tenir compte en interne

Jour 4 : algorithmie plus avancée: moteur de recommandation, produit similaire

Jour 5 : à définir

Détails des exercices

Jour 1 : introduction, algorithmie de base, manipulation de listes

Matin :

  • Exercice 1 : codage d'une fonction "afficher les éléments d'une liste" (4 lignes)
  • Exercice 2 : codage d'une fonction toString (4 lignes)
  • Exercice 3 : minimum et maximum d'une liste non triée (4 lignes)
  • Exercice 4 : trouver l'ensemble des éléments compris dans une fourchette de prix (6 lignes)
  • Exercice 5 : détecter si il existe au moins deux éléments ayant des prix identiques (6 lignes)
  • Exercice 6 : trouver l'ensemble des éléments ayant un prix identiques sur une liste non triée

Après-midi :

  • Exercice 7 : codage d'une recherche dichotomique
  • Exercice 8 à 11 : réécrire les exercices 3 à 6 de façon efficace sur une liste triée
  • Exercice 12 et + : implémentation de tris
