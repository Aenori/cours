Intégration

Vous travaillez sur un code qui comprend un ensemble de fonctionnalité et qui est appelé à être appelée par une autre application.
Ici l'autre application est une solution e-commerce, vous apportez des fonctionnalités avancées.
L'autre application est donc sous la responsabilité d'une autre équipe, voir d'une autre société.

Comment l'intégrez, ie comment faire en sorte que l'application e-commerce puisse appeler votre application ?

Plusieurs solutions d'intégration

  • webservice
  • executable
  • ESB (entreprise Serial Bus)
  • DLL / jar
